##
##### ctrl + (n)번 창으로 이동 -  작업 창에 ctrl + shift + (n) [n=1~9]
##### 작업창 전부 닫기  -  오른쪽 클릭 --> windows --> 모두닫기
##
##### ctrl + e : 프로젝트 파일 저장 (YYP 파일 1개 파일로 저장됨)
##### ctrl + shift + f -> 검색 및 바꾸기 : 코딩한 단어 찾기 / 변경
##### 레이아웃 : 인터페이스 설정 저장 및 불러오기
